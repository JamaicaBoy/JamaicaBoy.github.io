大模型辅助的工艺参数推荐（知识库增强 RAG）LLM-assisted Parameter Recommendation (Knowledge-augmented RAG)

本人参与Contributor中文核心《精密成形工程》· 已发表CSCD-core (J. Netshape Forming Eng.) · publishedRAG · 本地大模型RAG · local LLM

针对注射成形工艺调试“依赖经验、试错成本高、周期长”的问题，构建知识库增强(RAG)的生成式大模型工艺优化框架：自动搜集 819 篇中英文文献 → 切分 + 嵌入成向量知识库 → 本地部署大模型按“实际与目标质量偏差”动态生成参数调整建议并给出推理依据；在 PC、PMMA 等材料上 3 轮内精准收敛至目标质量，显著优于无知识库大模型与人工试错（需 7 轮以上）。For molding tuning that is experience-dependent, costly and slow, this builds a knowledge-augmented (RAG) generative-LLM framework: auto-collect 819 Chinese/English papers → chunk + embed into a vector knowledge base → a locally deployed LLM generates parameter adjustments with reasoning based on the gap between actual and target quality; on PC, PMMA and more it converges to target within 3 rounds, far better than an LLM without the knowledge base or manual trial-and-error (7+ rounds).

背景Background

注塑调参高度依赖老师傅经验，试错成本高、周期长，新人上手慢。Molding tuning relies heavily on veteran experience—costly, slow, and hard for newcomers.

方法Method

  • 自动搜集 819 篇中英文工艺文献 → 文档切分 → 嵌入模型 → 向量知识库。Auto-collect 819 papers → document chunking → embedding model → vector knowledge base.
  • 本地部署生成式大模型，按质量偏差检索增强生成参数调整建议与推理依据（Zotero / AnythingLLM / Ollama）。Locally deployed LLM does retrieval-augmented generation of parameter advice with reasoning, by the quality gap (Zotero / AnythingLLM / Ollama).

结果Results

  • PC、PMMA 等材料 3 轮内收敛至目标质量；无知识库大模型与人工试错（7 轮+）均不及。Converges within 3 rounds on PC/PMMA; beats the no-knowledge-base LLM and manual trial-and-error (7+ rounds).
  • 成果发表于中文核心《精密成形工程》。Published in the CSCD-core Journal of Netshape Forming Engineering.
